About this theme

a theme for people who want an uncluttered layout, but also have a lot to say! check the code for full instructions and notes.

  • this theme uses CSS variables to control your settings.
  • the title font supports english and japanese (but not kanji)
  • the header dimensions are intended for a 4:3 ratio image.
  • you can use the modal-img class to include right-aligned images in your modals.
  • this preview does not use captions for the modal icons, but you can add them by searching for caption 1 and caption 2 and deleting the HTML comment tags (<-- -->)
if you have any questions, you can email me at nenrikido@pm.me

welcome to the sakura page theme!

click the icons to the left and right to open the modals (pop-up boxes). the left modal displays information about this theme and the right contains a list of credits.

this box grows as you type, and once you hit the max height, it will automatically scroll. the same is true for the modal boxes.
